# RaaS SDK Simulator

A mobile-first RaaS (Remittance-as-a-Service) UI simulator built with Vite, React, and Tailwind CSS. It models onboarding, recipient, and transaction flows with animated UI components.

## Quick start

```bash
npm install
npm run dev
```

Open the URL printed by Vite (usually `http://localhost:5173`).

## Scripts

- `npm run dev` / `npm run start` — start the dev server
- `npm run build` — production build
- `npm run preview` — serve the production build locally
- `npm run lint` — ESLint checks
- `npm run typecheck` — TypeScript type check
- `npm run test` — Vitest smoke tests

## Project structure

- `index.html` — Vite entry HTML
- `src/main.tsx` — React entrypoint
- `src/app` — UI flows and components
- `src/styles` — Tailwind + theme styles

## Requirements

- Node.js 22+ (project verified on Node 22.14.0)
